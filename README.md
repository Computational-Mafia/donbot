# Donbot
 Donbot is a library and associated set of utilities for power users to automate operations on MafiaScum.Net. Originally based on `modbot`, a project aimed at automating game moderation activities, this effort focuses on accessible automation of simple, trigger-based activities relevant to a broad variety of contexts.

 ## The Donbot Library
 Donbot utilities are based on our Donbot function library aimed at simplifying utility development and customization. It wraps a variety of powerful functions automating simple forum operations that can be flexibly sequenced to perform a wide variety of activities. Create an instance of the Donbot class with your username and password (and potentially other parameters), and you'll be able to do things like:

- Collect a range of posts from a thread
- Make posts in a specified thread with specified content
- Send pms to a user with a specified subject and body
- Collect the number of posts in a specified thread
- Collect id matching a specified scummer's username

 Please don't use these functions haphazardly, especially those that make posts or send PMs, as misuse thereof can be against Site Rules, get you banned, and most importantly cause trouble for a lot of decent people.

Within its directory, we include a detailed usage tutorial intending that even Python beginners can quickly begin applying the library. Please let someone know if it's wrong or unclear, because it'll always be a work in progress.
